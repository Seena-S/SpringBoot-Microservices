# SpringBoot-Microservices
This repository is part of my Microservices Learning Series, where I build and explore microservices architecture step-by-step using Spring Boot.
The first project in this series is a Quiz Application, starting with the Question Service. This service is responsible for managing quiz questions and exposes REST APIs to perform CRUD operations and retrieve questions by category. It is designed to be integrated later with other microservices in the quiz platform.

Features
========
Add new quiz questions
Retrieve all questions
Filter questions by category
Update questions by ID
Delete questions by ID

Technologies Used
=================
Java
Spring Boot
Spring Data JPA
REST APIs
PostgreSQL
Maven
